**OBJETIVO:** Dejar de hacer tareas repetitivas "clic a clic". Aprenderás a escribir scripts que crean planos automáticamente, exportan archivos en masa y generan geometría paramétrica compleja.

:::tip[Requisito Previo]
No necesitas ser un experto programador, pero entender la lógica básica de **"Si... Entonces" (If/Else)** y **"Bucles" (Loops)** te ayudará muchísimo.
:::
